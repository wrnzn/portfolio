# Joseph Alejo Portfolio

Recruiter-first portfolio for software, web, and applied AI internship roles.

## Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Local Development

```bash
npm install
npm run dev
```

## Quality Checks

```bash
npm run build
npm run lint
npx vitest run
npm audit --audit-level=moderate
```

## Content Sources

- Portfolio copy is curated in `src/data/portfolioData.ts`.
- Latest resume source is preserved in `src/data/resume-source.json`.
- Resume download lives in `public/Joseph_Alejo_Resume.pdf`.
- Project screenshots live in `public/project-shots/`.
- Sanitized public project documentation lives in `public/docs/`.
